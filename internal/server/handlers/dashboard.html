使用说明

  • API Key 就是 .env 中的 API_KEY。
  • 所有请求都会带上 ?api_key= 参数；HTTPS 可防止中间人。
  • 本面板只调用已有的 REST API，不会绕过权限校验。
  • 页面不依赖任何外部 CDN，支持纯内网部署。
